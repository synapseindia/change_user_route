INTRODUCTION
------------

Change User Route module is used to change the path of login , password , logout and registration .


REQUIREMENTS
------------

 -No special requirements.

INSTALLATION
------------

 * Module: Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/documentation/install/modules-themes/modules-8
   for further information.

CONFIGURATION
-------------

After the installation of module  admin can configure login , password and registration path from User Interface Change user Route section .
